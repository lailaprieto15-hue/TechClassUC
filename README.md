# TechClassUC - Sistema de Gestión de Atención

Sistema de gestión de turnos y atención al cliente desarrollado en Java.

## Características
- Gestión de colas de clientes.
- Historial de atención.
- Interfaz gráfica moderna con Swing.
- Funcionalidad de deshacer acciones.

## Tecnologías
- Java 24
- Maven
- JUnit 5
